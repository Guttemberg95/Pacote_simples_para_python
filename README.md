# calcula_python

Description. 
The package calcula_python is used to:

	- Somar
	- Subtrair
    - Dividir
    - Mutiplicar

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install calcula_python

```bash
pip install calcula_python
```

## Usage

```python
from calcula_python import calcula
calcula.my_function()
```
